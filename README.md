# CodeAlpha_PhishingAwareness

# 🎣 CodeAlpha — Phishing Awareness Training
> **CodeAlpha Cybersecurity Internship | Task 2**

An interactive, browser-based phishing awareness training module built with **HTML, CSS, and JavaScript**. Educates users on recognizing phishing emails, fake websites, and social engineering tactics through real-world examples and an interactive quiz.

---

## 📸 Features
- 🧠 Explains phishing attack types (email, spear phishing, smishing, vishing)
- 🔍 Teaches users how to spot red flags in emails and fake websites
- 💡 Real-world examples and case studies
- ✅ Interactive quiz to test understanding
- 📱 Fully responsive — works on desktop and mobile
- 🚀 Zero dependencies — runs directly in any browser

---

## 🚀 Usage

No installation needed! Simply open the file in your browser:

```bash
# Option 1: Double-click the file
phishing_awareness.html

# Option 2: Open via terminal
open phishing_awareness.html        # macOS
xdg-open phishing_awareness.html    # Linux
start phishing_awareness.html       # Windows
```

## 📚 Topics Covered
| Topic | Description |
|-------|-------------|
| What is Phishing? | Definition and overview of phishing attacks |
| Types of Phishing | Email, Spear, Smishing, Vishing, Clone phishing |
| Red Flags | How to identify suspicious emails and URLs |
| Social Engineering | Tactics attackers use to manipulate victims |
| Best Practices | Tips to stay safe online |
| Quiz | Interactive knowledge check |
